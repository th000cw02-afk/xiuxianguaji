# 文字修仙 — 产品定位

## 是什么

**文字修仙** 是一套开源的「纯文字 + 挂机数值」游戏框架。本仓库（修仙挂机）是该框架下的**修仙题材实例 Demo**。

| 层级 | 名称 | 说明 |
|------|------|------|
| 框架 | 文字修仙 | 玩法范式：MUD 终端 UI + WebView + JS 数值引擎 |
| 实例 | 修仙挂机 | 当前仓库的题材皮肤与数值配置 |
| 技术 | Android + assets | 单机、localStorage 存档、无联网 |

## 目标受众

- 喜欢 MUD/终端风、纯文字呈现的玩家
- 希望 Fork 后快速换皮（江湖、星际、赛博等）的独立开发者
- 需要完整挂机数值参考的开源学习者

## 与同类产品的差异

| 维度 | 文字修仙 | 常见同质化产品 |
|------|----------|----------------|
| 界面 | ASCII/终端框、VT323 字体 | 立绘、卡牌、弹窗 UI |
| 随机 | 气运偏移体系（返虚路层数） | 纯 Math.random |
| 中期 | 化身平行世界 + 内天地世界攻略 | 单一角色线性推图 |
| 自动化 | 10+ 可配置自动开关 | 有限或付费加速 |
| 开源 | MIT，theme 配置可 Fork | 多为闭源 |

## 核心体验循环

```
修炼吸灵 → 探索挂机 → 洞府/法则/炼丹/法宝养成
    → 返虚路提气运 → 化身/内天地深化 → 长期数值成长
```

新增 **纪事 / 道果 / 天机阁** 三层，分别提供：叙事反馈、长期目标、每日短期契约。
