# 游戏系统全景

本文梳理《修仙挂机》实例中已实现的子系统，供设计与二次创作参考。公式细节见 [游戏说明.txt](../app/src/main/assets/游戏说明.txt)。

## 界面 Tab 与子系统

| Tab | 系统 | 核心循环 |
|-----|------|----------|
| 修炼 | 境界、灵力、神通 | 吸灵力 → 升级/突破 → 解锁神通 |
| 探索 | 99 地图 + 无尽虚空 | 挂机战斗 → 掉落 → 推图 |
| 洞府 | 灵脉/仙脉、聚灵阵、灵田 | 投入资源 → 提升产出 |
| 法则 | 五法则加点 | 碎片 → 法则等级 → 属性倍率 |
| 炼丹 | 丹炉、配方 | 灵药 → 丹药 → 突破/属性 |
| 法宝 | 悟道珠/青木剑 | 天材地宝 → 位格/暴击 |
| 纳戒 | 背包 | 囤积 → 自动/手动使用 |
| 化身 | 最多 4 角色 | 100 本源创建 → 平行养成 → 吸收 |
| 设置 | 存档、自动开关 | 长期挂机配置 |

## 嵌套大系统（无独立 Tab）

| 系统 | 入口 | 说明 |
|------|------|------|

## 随机数体系

- `Math.random()`：气运偏移后，用于战斗/掉落（利于玩家）
- `Math.randomNeutral()`：中性随机（命名、日志等）
- 气运 = 返虚路层数，系数 `1 / (1 + 气运/100)`

详见 [随机数说明文档.md](../随机数说明文档.md)。

## 存档结构

- 每角色独立 `localStorage` 键
- 主角存档共享：返虚路、本源、纪事/成就/天机阁、仙脉之灵技能等级
- 多角色列表：`wenziguaji_characters`
