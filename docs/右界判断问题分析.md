# Math.random() 右界判断问题分析

> **核心问题**：当前气运系统通过压缩随机数值域实现，导致使用**右界判断**的场景出现bug
> 
> **气运影响机制**：
> - `Math.random()` 返回值 = 原随机数 × 系数
> - 气运100时，系数 = 0.5，Math.random() 范围变成 [0, 0.5)
> - **左界判断**（`Math.random() < p`）：正常工作 ✅
> - **右界判断**（`Math.random() >= p`）：出现bug ❌

---

## 🔴 问题场景（使用右界判断）

### 1. 敌人出手判定 - ❌ 严重问题

**位置**：
- [game.js:14506](../app/src/main/assets/game.js#L14506)
- [game.js:14620](../app/src/main/assets/game.js#L14620)

**代码**：
```javascript
const actChance = enemyAttacker.speed / 100;
if (Math.random() >= (1 - actChance)) {
    // 敌人出手
}
```

**问题分析**：
| 敌人速度 | 判定条件 | 原本概率 | 气运100实际概率 | 问题 |
|---------|---------|---------|----------------|------|
| 80% | >= 0.2 | 80% | 60% | ❌ 概率降低不足 |
| 60% | >= 0.4 | 60% | 60% | ❌ 概率未变 |
| 50% | >= 0.5 | 50% | 0% | ❌ **永远无法出手** |
| 40% | >= 0.6 | 40% | 0% | ❌ **永远无法出手** |
| 20% | >= 0.8 | 20% | 0% | ❌ **永远无法出手** |

**计算示例**（气运100，Math.random()最大0.5）：
- 速度80%：判定 `Math.random() >= 0.2`
  - 实际概率 = (0.5-0.2)/0.5 = 60%（应该是40%）
- 速度40%：判定 `Math.random() >= 0.6`
  - 但最大只有0.5，**永远不成立**

**修复方案**：
```javascript
// 改成左界判断
if (Math.random() < actChance) {
    // 敌人出手
}
```

### 2. 技能数量判定 - ⚠️ 轻微问题

**位置**：[game.js:13486-13498](../app/src/main/assets/game.js#L13486)

**代码**：
```javascript
const rand = Math.random() * 100;
if (isEnemy) {
    if (rand < 13) return 1;   // ✅ 左界
    if (rand < 25) return 2;   // ✅ 左界
    if (rand < 37) return 3;   // ✅ 左界
    if (rand < 53) return 4;   // ✅ 左界
    return 5;                   // ❌ 隐式右界：rand >= 53
}
if (rand < 13) return 5;       // ✅ 左界
if (rand < 25) return 4;       // ✅ 左界
if (rand < 37) return 3;       // ✅ 左界
if (rand < 53) return 2;       // ✅ 左界
return 1;                       // ❌ 隐式右界：rand >= 53
```

**问题分析**（气运100，rand范围 [0, 50)）：
| 场景 | 原本概率 | 气运100实际概率 | 问题 |
|------|---------|----------------|------|
| 敌人返回5技能 | 47% | 0% | ❌ 永远无法获得5技能 |
| 玩家返回1技能 | 47% | 0% | ❌ 永远无法获得1技能 |

**影响程度**：较轻微，因为：
- 敌人少一个技能对玩家有利
- 玩家少一个技能影响不大（通常有多个技能）

**修复方案**（可选）：
```javascript
const rand = Math.random() * 100;
if (isEnemy) {
    if (rand < 13) return 1;
    if (rand < 25) return 2;
    if (rand < 37) return 3;
    if (rand < 53) return 4;
    if (rand < 100) return 5;   // ✅ 明确左界
}
```
